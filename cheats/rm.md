# rm

## Basic removal

- `rm <file>` — Remove a file
- `rm <file1> <file2>` — Remove multiple files
- `rm -i <file>` — Prompt before every removal

## Forced and recursive removal

- `rm -f <file>` — Force removal without prompt
- `rm -r <dir>` — Remove a directory and its contents recursively
- `rm -rf <dir>` — Force recursive removal without prompt
- `rm -d <dir>` — Remove an empty directory
