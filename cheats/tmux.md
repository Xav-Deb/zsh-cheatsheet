# tmux

## Sessions

- `tmux new -s <name>` — Create session
- `tmux ls` — List sessions
- `tmux attach -t <name>` — Attach session

## Panes

- `Ctrl-b %` — Split vertically
- `Ctrl-b "` — Split horizontally
- `Ctrl-b o` — Next pane

## Windows

- `Ctrl-b c` — New window
- `Ctrl-b n` — Next window
