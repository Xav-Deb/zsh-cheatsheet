# cp

## Copying files

- `cp <source> <dest>` — Copy a file to a destination
- `cp <file> <dir>/` — Copy a file into a directory
- `cp -i <source> <dest>` — Prompt before overwriting

## Copying directories

- `cp -r <dir_src> <dir_dest>` — Copy a directory recursively
- `cp -a <dir_src> <dir_dest>` — Archive mode (copy recursively and preserve attributes)
- `cp -p <source> <dest>` — Copy and preserve permissions and timestamps
