# mv

## Moving and renaming

- `mv <source> <dest>` — Move or rename a file/directory
- `mv <file> <dir>/` — Move a file into a directory
- `mv -i <source> <dest>` — Prompt before overwriting an existing file
- `mv -n <source> <dest>` — Do not overwrite any existing file
- `mv -v <source> <dest>` — Verbose mode, explain what is being done
