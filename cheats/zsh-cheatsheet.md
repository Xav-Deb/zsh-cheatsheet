# zsh-cheatsheet

## Usage

- `^H` — Open help for the current command (Ctrl+H)
- `zsh-cheatsheet update` — Synchronize cheat sheets from GitHub
- `zsh-cheatsheet upgrade` — Check for updates and upgrade interactively
- `zsh-cheatsheet reload` — Reload plugin functions safely

## Maintenance

- `~/.cache/zsh-cheatsheet/cheats` — Directory where downloaded cheats are stored
- `zsh-cheatsheet-check-updates` — Internal function for periodic update checks
