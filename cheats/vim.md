# vim

## Modes

- `i` — Insert mode
- `Esc` — Normal mode
- `:` — Command mode

## Navigation

- `h j k l` — Move cursor
- `gg` — Top of file
- `G` — Bottom of file

## Editing

- `dd` — Delete line
- `yy` — Yank line
- `p` — Paste
- `u` — Undo last action
- `Ctrl + r` — Redo

## Search & Replace

- `/[query]` — Search for query
- `n` — Next match
- `N` — Previous match
- `:%s/old/new/g` — Replace 'old' with 'new' in whole file
- `:%s/old/new/gc` — Replace with confirmation

## Splits & Tabs

- `:sp [file]` — Split horizontally
- `:vsp [file]` — Split vertically
- `Ctrl + w [h/j/k/l]` — Navigate between splits
- `:tabnew` — Open a new tab
